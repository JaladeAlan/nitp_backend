# NITP Oyo State Chapter Portal

This project is the official web portal for the Nigerian Institute of Town Planners (NITP) Oyo State Chapter. It provides news, events, gallery, projects, resources, partners, and a secure admin panel. The portal consists of a Laravel backend API and a React frontend.

---

## Features

- User authentication with JWT tokens  
- Email verification and password reset  
- Public content: News, Events, Projects, Resources, Partners  
- Admin panel for managing all content and viewing dashboard stats  
- File uploads for images and PDFs  
- Fully documented API with example responses  
- Responsive React frontend with reusable components  

---

## Tech Stack

- Backend: Laravel 10, PHP 8+, MySQL/PostgreSQL  
- Frontend: React 18, TailwindCSS, Axios, React Router  
- Authentication: JWT via `tymon/jwt-auth`  
- API Documentation: OpenAPI 3.1 YAML  
- File Storage: Local or cloud
